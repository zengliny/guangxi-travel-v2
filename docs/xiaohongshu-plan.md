## 二、用户画像

### 画像1：北海游客（25-40岁）

- **职业**：都市白领、带薪休假
- **出行方式**：自由行、情侣/闺蜜
- **预算**：人均1000-3000元
- **偏好**：拍照打卡、美食探店、深度体验
- **痛点**：不想踩坑，想要真实评价
- **活跃时间**：下班后、周末、节假日

### 画像2：养老买家（45-65岁）

- **背景**：退休人士/候鸟老人/投资客
- **需求**：养老度假/长线投资/避寒过冬
- **预算**：30-150万房产预算
- **偏好**：环境宜居、医疗便利、生活配套
- **痛点**：担心被坑，需要真实信息
- **活跃时间**：上午、下午（时段充裕）

### 画像3：亲子游家庭（30-45岁）

- **构成**：一家三口/四口
- **需求**：周末短途/寒暑假长线
- **预算**：人均500-2000元/天
- **偏好**：安全、省心、有趣、教育性
- **痛点**：孩子开心+大人轻松
- **活跃时间**：周末、寒暑假

## 四、爆款标题公式

### 公式1：数字+痛点+解决方案
```
数字+具体痛点 → 【解决方案���
例：3个北海旅游必踩的坑，第2个最多人上当
```

### 公式2：反常识+好奇
```
颠覆认知的事实 → 引发好奇
例：北海本地人从来不去这些景点
```

### 公式3：本地人/业内人身份
```
身份标签 → 权威感
例：涠洲岛民宿老板揭秘：这4类房型千万别订
```

### 公式4：对比+悬念
```
对比反差 → 悬念
例：同样是去北海玩，为什么她花的钱只有你的一半？
```

### 公式5：结果导向+利益点
```
明确好处 → 利益驱动
例：这家民宿住完睡不着，真的太美了
```

### 公式6：避坑/防骗类
```
恐惧心理 → 规避损失
例：来北海前必看！否则多花冤枉钱别怪我没提醒
```

### 公式7：盘点/清单类
```
归纳整理 → 收藏价值
例：北海必吃的10家店，最后一家本地人私藏
```

## 八、执行建议

### 1. 内容优先级
1. **先做避坑类**（建立信任，积累粉丝）
2. **再做美食/景点类**（高频需求，扩大曝光）
3. **然后亲子/养老类**（精准人群，深度转化）
4. **最后房产类**（高客单价，变现核心）

### 2. 爆款打造技巧
- 首图：用对比图/惊艳图/痛点图
- 开头：3秒内抛出亮点或痛点
- 结构：结论先行+案例支撑+总结互动
- 结尾：引导评论、收藏、关注

### 3. 数据复盘
- 发布后24小时看曝光
- 72小时看互动率
- 1周后优化策略
